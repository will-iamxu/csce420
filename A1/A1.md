# Assignment 1: Search in Pacman

Students implement **depth-first search**, **breadth-first search**, **uniform cost search**, and **A*** search algorithms. These algorithms are used to solve navigation and traveling salesman‚Äìstyle problems in the Pacman world.

In this assignment, your Pacman agent will find paths through his maze world‚Äîboth to reach a particular location and to collect food efficiently. You will build general search algorithms and apply them to Pacman scenarios.

## Welcome to Pacman

After downloading and unzipping the code, you can play Pacman with:

```bash
python pacman.py
```

Pacman lives in a maze of corridors and food pellets. Navigating efficiently is the key challenge.

### Example Agent

The simplest agent is `GoWestAgent`, which always moves west:

```bash
python pacman.py --layout testMaze --pacman GoWestAgent
```

It fails when turning is required:

```bash
python pacman.py --layout tinyMaze --pacman GoWestAgent
```

Exit the game with **CTRL+C** if Pacman gets stuck.
